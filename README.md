# Tiny C Projects

The repository was originally based on *Tiny C Projects* by Dan Gookin.

However, soon I realized, that I do not like to have a lot of repositories to work in as my main goal is to (re)learn C. For this journey, I will use multiple sources of exercises and will store all these in one repository in corresponding subfolders.

Here is the current list of the content folders and their origin:

* `tiny_c_projects` contain exercises from *Tiny C Projects*  by Dan Gookin
* `modern_c` contains exercises from *Modern C, Third Edition* by Jens Gustedt
* `orso` contains exercises from both *Programozz Te is mesterfokon! -- Munkafüzet I.* and *Programozz Te is mesterfokon! -- Minden nap!* by Lajos Orsó
* `dynamic_programmin` contains exercises from *The Dynamic Programming Manual* by Gabor L. Hajba (yes, that's me)

You may see that not all folders mentioned above are present currently. But it represents my current development plan and soon enough you will encounter all of them.
